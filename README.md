# Arena Fox

A modern, animated login experience for Arena Fox built with Next.js (App Router), TypeScript, and Framer Motion. The UI layers aurora-inspired gradients, glassmorphism, and micro-interactions to create an immersive authentication portal that's ready for Vercel deployment.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Build for production (Vercel compatible):
   ```bash
   npm run build
   npm start
   ```

The primary page lives in `app/page.tsx`, with styling in `app/page.module.css` and the animated fox glyph in `app/components/FoxGlyph.tsx`.
